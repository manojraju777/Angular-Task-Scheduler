Schedule your tasks as below

Scheduler contains 6 stars as the below chart

                # ┌────────────── second (optional)
                # │ ┌──────────── minute
                # │ │ ┌────────── hour
                # │ │ │ ┌──────── day of month
                # │ │ │ │ ┌────── month
                # │ │ │ │ │ ┌──── day of week
                # │ │ │ │ │ │
                # │ │ │ │ │ │
                # * * * * * *

First star represents the seconds, second is for minutes, third for hours, fourth for day of month, fifth for month and 6th for day of week

This means you can setup a schedule for literally any time. Maybe once a year? Or every Tuesday at 8am in January and July. There's really no limitation

Samples

To run a scheduler everyday at 18:00, Set it as 0 18 * * *

To run a scheduler everyday at 04:05, Set it as 5 4 * * *

To run a scuduler on every friday at 07:00, Set it as 0 7 * * 5
